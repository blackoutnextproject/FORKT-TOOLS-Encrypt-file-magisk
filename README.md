🔐 FORKT-TOOLS Encrypt File Magisk

Simple tool for encrypting Magisk configuration files directly from Termux.

---

✨ Features

- 🔒 Encrypt Magisk configuration files
- ⚡ Fast and lightweight
- 🐍 Python-based
- 📱 Optimized for Termux
- 🛠 Easy installation

---

📦 Installation

```apt update && apt upgrade -y && apt install git python -y```

```git clone https://github.com/blackoutnextproject/FORKT-TOOLS-Encrypt-file-magisk.git && cd FORKT-TOOLS-Encrypt-file-magisk```
```bash req.sh && python3 kt.py```

---

📋 Requirements

- Termux
- Python 3
- Git

python3 kt.py
