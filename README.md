FORKT-TOOLS Encrypt File Magisk

Tool sederhana untuk mengenkripsi file konfigurasi Magisk dan mengelola output terenkripsi.

Installation

Jalankan perintah berikut di Termux:

apt update && apt upgrade -y
apt install git python -y
git clone https://github.com/blackoutnextproject/FORKT-TOOLS-Encrypt-file-magisk.git
cd FORKT-TOOLS-Encrypt-file-magisk
bash req.sh
python3 kt.py

Features

- Encrypt file Magisk
- Simple CLI interface
- Python-based tool
- Easy Termux installation

Requirements

- Termux
- Python 3
- Git
